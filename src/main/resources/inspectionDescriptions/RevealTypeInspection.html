Displays the type of the expression passed as argument to reveal_type(). Also reports incorrect calls. 

a: str
reveal_type(a)  # Revealed type is "str"

# `reveal_type()` expects exactly one positional argument
reveal_type()
reveal_type(lorem = b'ipsum')
reveal_type(42, foo = 'bar')

The "severity" option has no effect. The two types of subinspections are always reported as weak warnings and warnings, correspondingly.
